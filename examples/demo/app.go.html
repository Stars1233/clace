Clace Demo Apps

These apps showcase the features of Clace. All the apps are running on a single Clace server. The code for the apps is here

Disk Usage

This app shows the disk usage on the node, allowing the user to find which directory is consuming the most space. It is a wrapper over the "du" command and shows simple Hypermedia based transitions.

Open Disk Usage App

Memory Usage

This app shows the resident memory usage, aggregated by parent process. The data comes from the "ps" command. The "plot" ui library is used as an ESM module to show the data visualization.

Open Memory Usage App

CowBull Word Game

This app builds the UI for a CowBull word finding game. It demonstrates the use of more complex Hypermedia controls and integration with Tailwind CSS and DaisyUI.

Open Cowbull App
This page itself is a static Clace app. The complete installation steps for the service and apps are below.

First, Install Clace. Then enable auto TLS certificates. Then start the server and install the apps.

$ nohup clace server start &
$ clace app create --domain utils.demo.clace.io --auth_type=none --approve /memory_usage ~/clace_source/clace/examples/memory_usage/
$ clace app create --domain utils.demo.clace.io --auth_type=none --approve /disk_usage ~/clace_source/clace/examples/disk_usage/
$ clace app create --domain cowbull.demo.clace.io --auth_type=none --approve / ~/clace_source/clace/examples/cowbull
$ clace app create --auth_type=none --approve / ~/clace_source/clace/examples/demo
